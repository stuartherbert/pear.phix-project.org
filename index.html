Phix Project PEAR channel

Using this Channel

This channel is to be used with the PEAR installer.

Registering the channel: 
pear channel-discover pear.phix-project.org
Listing available packages: 
pear remote-list -c phix
Installing a package: 
pear install phix/package_name
Installing a specific version/stability: 
pear install phix/package_name-1.0.0
pear install phix/package_name-beta
Receiving updates via a feed: 
http://pear.phix-project.org/feed.xml
